# BeeCollector
 
Welcome to my Smart BeeHive project!

## Goals

Build a smart hive that has integrated sensors!

* Check Tempurature around the hive
* Check Humidity
* Weight the hive
	* Measure honey production in the summer
	* Measure hive Health/food in the winter
